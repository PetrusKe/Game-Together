2.服务器
    主线程：服务器开启运行
    子线程：每一个手机用户接入就开启一个子线程，接收手机端数据

    主线程退出：执行完毕自然退出
    子线程退出：设置一个标识变量（如:running），如果标识变量变化，则跳出子线程的循环使得子线程结束

    主动断开所有客户端连接后关闭服务
